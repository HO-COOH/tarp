Detailed Description

The abstract MakefileGenerator class, which shall be inherited by specific makefile generator.

To create a makefile generator for a specific build system, any subclasses should

  1. inherit this class
  2. call ProjectGenerator::registerMakefileGenerator("<generator name>", FactoryFunction); where the FactoryFunction accepts a const reference to a ProjectDirCreator object and a const reference to a CodeGenerator object and returns a std::unique_ptr of this class, ie. to have this signature: std::unique_ptr<MakefileGenerator> FactoryFunction(ProjectDirCreator const&, CodeGenerator const&); For example, a CMakeGenerator
    class CMakeGenerator: public MakefileGenerator
    {
    private:
    static bool registered = ProjectGenerator::registerMakefileGenerator("cmake",
    [](ProjectDirCreator const& projectDir, CodeGenerator const& code) -> std::unique_ptr<MakefileGenerator>
    {
    return std::make_unique<CMakeGenerator>(projectDir, code);
    });
    public:
    //...
    };
  3. Provide the definition of this virtual function:
    void writeMakefile();
